1: Een git opzetten in VSO

Gedurende dit keuzevak maken we gebruik van de HoloLens, ontwikkeld door Microsoft, de programmeertaal C#, ontwikkeld en onderhouden door Microsoft, Visual Studio, ontwikkeld door Microsoft. Dus we gaan ook gratis git repository van Microsoft gebruiken. Deze heet Visual Studio Online. Visual Studio zelf heeft een naadloze integratie met VSO. Je mag ook gewoon GitHub of een andere git gebruiken, maar dat wordt niet door de docenten ondersteund.
