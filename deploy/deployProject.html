Deployen van je Applicatie

Deze pagina beschrijft het bekijken van je HoloLens applicatie in zowel de emulator als op de HoloLens. We beginnen met de Emulator.

Deployen naar je emulator

Hieronder beschrijven we in stappen hoe je jouw HoloLens applicatie kan bekijken in de HoloLens emulator. Het deployen van de applicatie moet een twee eisen voldoen. HoloLens werkt alleen met het Microsoft Store platform. Daarnaast moet het deployen gebeuren via Visual Studio.

  1. We drukken op File -> Build Settings of Ctrl + Shift + B. Als eerste drukken we daar op Add Open Scenes. Selecteer bij Platform Universal Windows Platform.

Slot

Hier eindigen we dit hoofdstuk. In het volgende hoofdstuk gaan we je laten zien hoe je de applicatie in de HoloLens Emulator en op de HoloLens kunt laten draaien.
